# CSI142-VILLAGE-CLINIC-QUEUE-MANAGER
This project will make a Java console-based program that keeps track of how many patients are in a local village health clinic. A receptionist will be able to register patients, place them in a queue based on their needs, and keep track of their progress. 


# How to build project 
javac src/com/clinic/queue/*.java

# How to run project 
java -cp src com.clinic.queue.Main
